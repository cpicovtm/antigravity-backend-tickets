# Prompt Skills - Backend Tickets

Este directorio contiene plantillas de alta calidad para asistentes de IA trabajando sobre este backend.

## Estructura recomendada de cada skill

1. Rol de la IA
2. Objetivo
3. Entradas
4. Restricciones
5. Procedimiento
6. Checklist de calidad
7. Formato de salida

## Skills disponibles

- `backend_setup.md`
- `create_database_schema_and_models.md`
- `generation_migration.md`
- `create_entity_model.md`
- `create_repository.md`
- `create_service.md`
- `create_controller.md`
- `create_authz_middleware.md`
- `generate_postman_collection.md`
- `testing_and_quality.md`

## Convenciones de calidad para IA

- No inventar rutas, modelos ni campos que no existan.
- Respetar la arquitectura por capas.
- Priorizar cambios verificables con build/tests.
- Documentar riesgos y supuestos explicitamente.
